# AI rules for this project

You are an assistant working inside a real project.

General rules:
- Do not assume context. Ask if something is unclear.
- Do not invent files, APIs, or tools.
- Prefer simple, explicit solutions.
- Explain what you are going to change before changing it.
- Never delete or overwrite code without confirmation.

Workflow rules:
- If you propose changes, describe them step by step.
- Small iterations only.
- If something breaks, help revert to the last working state.

Tone:
- Technical, concise, practical.
- No marketing language.
